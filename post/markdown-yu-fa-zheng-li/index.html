Markdown语法整理

2023-09-05 4 min read # 笔记

Markdown个人笔记于2023/09/05整理

块级元素

占位空间：一行及以上
包括：标题、引用、列表等

1. 标题

  • 语法：'#'
  • 用法如下：
    # 标题1展示
    ###### 标题6展示

  • 效果展示：

    标题1展示

    标题6展示

2. 引用

  • 语法：'>'
  • 用法如下：
> 引用展示
  • 效果展示：

    引用展示

3. 有序列表

  • 语法：'序号.'
  • 用法如下：
1. 第一列
2. 第二列
  • 效果展示：
    1. 第一列
    2. 第二列

4. 无序列表

  • 语法：'-'
  • 用法如下：
- 第一列
- 第二列
  • 效果展示：
    • 第一列
    • 第二列

5. 任务列表

  • 语法：'- [ ] '或' - [x] '
  • 用法如下：
- [ ] 第一列
- [x] 第二列
  • 效果展示：

6. 代码块

  • 语法：'``` 语言'
  • 用法如下：
    ``` java    //开始标识
        int i = 0;
    ```     //结束标识
  • 效果展示：
        int i = 0;

7. 数学公式

  • 语法：'$$'
  • 用法如下：
    $$  //开始标识
       数学公式
    $$  //结束标识
  • 效果展示：

    ∂f∂x=2ax \frac{\partial f}{\partial x} = 2\sqrt{a}x ∂x∂f​=2a​x

8. 表格

  • 语法：'|(😃---(:)|'
  • 用法如下：
    |姓名|年龄|成绩|
    |:---|---:|:---:|
    |张三|19|99|
    |李四|20|81|
  • 效果展示：
    姓名 年龄 成绩
    张三 19 99
    李四 20 81

9. 脚注

  • 语法：'[^内容]'
  • 用法如下：
   一键三连[^三连]
   [^三连]:点赞、关注、收藏
  • 效果展示：
    一键三连[1]

10. 横线

  • 语法：'---'
  • 用法如下：
   ---
   你好
   ---
  • 效果展示：
你好

11. 链接

  • 语法：'标题'
  • 用法如下：
   [bilibili](https://www.bilibili.com "B站")
   [百度][id]
   请参考[链接](#11-链接)

   [id]:https://www.baidu.com "搜索引擎"
  • 效果展示：
    bilibili
    百度 百度 百度
    请参考链接

11. URL

  • 语法：'https://xxx'
  • 用法如下：
   https://www.bilibili.com
  • 效果展示：

https://www.bilibili.com

行内元素

占位空间：内容大小

1. 斜体

  • 语法：'**'
  • 用法如下：
  *斜体*
  • 效果展示：
    斜体

2. 粗体

  • 语法：'****'
  • 用法如下：
  **粗体**
  • 效果展示：
    粗体

3. 删除线

  • 语法：'~~~~'
  • 用法如下：
  ~~删除线~~
  • 效果展示：
    删除线

4. 代码行

  • 语法：'``'
  • 用法如下：
  `System.out.println()`
  • 效果展示：
    System.out.println()

5. 下划线

  • 语法：'<u></u>'
  • 用法如下：
  <u>下划线</u>
  • 效果展示：
    下划线

6. 数学函数

  • 语法：'$数学函数$'
  • 用法如下：
  $\theta=x^2$
  • 效果展示：
    θ=x2\theta=x^2θ=x2

7. 下标

  • 语法：'~'
  • 用法如下：
  H~2~O
  • 效果展示：
    H2O

8. 上标

  • 语法：'^'
  • 用法如下：
  x^2^
  • 效果展示：
    x2

9. 高亮

  • 语法：'==高亮内容=='
  • 用法如下：
  ==高亮内容==
  • 效果展示：
    高亮内容

10. HTML代码

  • 语法：'HTML标签'
  • 用法如下：
  <h1>我是标题1 h1</h1>
  • 效果展示：

我是标题1 h1
